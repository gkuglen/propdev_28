# Partnership Deal Calculator

A real estate development partnership calculator for tear-down / rebuild equity split deals.

## Features

- **Deal Calculator** — construction financing, seller-financed notes, partner returns
- **Owner Opportunity Cost** — partnership vs. selling as-is comparison
- **Traditional vs. Prefab** — construction method cost and return comparison

## Local Development

```bash
npm install
npm run dev
```

Opens at `http://localhost:5173`

## Deploy

Deployed via Vercel — push to `main` branch to trigger automatic redeploy.
